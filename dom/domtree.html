NodeList vs HTMLCollection

  • Keduanya merupakan kumpulan node
  • struktur datanya mirip array
  • nodeist dapat berisi node apapun sedangkan HTMLCollection harus berisi elemen HTML
  • HTMLCollection bersifat live sedangkan nodelist tidak

Struktur Hierarki DOM

  • Root Node
    • node yang menjadi sumber dari semua node lain di dlm DOM
    • document
  • Parent Node

    node yg berada 1 tingkat diatas node lain

    body adalh parent dari h1,p,p,div, dan a

  • Child Node

    node yg berada 1 tingkat dibawah node yg lain

    h1 adalah child dari body

    • ada juga sibling atau saudara dari child node

    , dll
